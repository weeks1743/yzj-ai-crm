# 云之家产品能力知识库

## 一、公司背景

- **公司**：云之家网络（重庆）有限公司
- **定位**：SaaS协同办公NO.1
- **市场份额**：IDC 2023H1数据，中国企业团队协同软件市场（云模式）份额62.5%，远超第二名微软13.4%
- **历史**：2011-2024年，13年专注SaaS协同
- **客户量**：10,000+客户，数百家500强和专精特新企业
- **连续领跑**：2015-2017中大型移动办公市场连续三年领跑；2018-2023连续六年领跑SaaS协同

## 二、产品理念

**赋能人人，敏捷协同**

四大核心能力：
1. 敏捷的场景协同能力
2. 智能的数据协同能力
3. 高效的组织协同能力
4. 可组装的业务协同能力

## 三、核心产品模块

### 3.1 统一门户

**解决问题**：异构系统多且杂，账号密码易遗忘，业务应用查找不便，人找"事"效率低

**核心能力**：
- 三端统一门户（Web/PC/APP），数据无缝切换
- 千人千面，按需创建多维业务门户（信息门户、流程门户、报表门户、学习门户、知识门户、党建门户等）
- 灵活卡片展现，信息聚焦直观
- 统一认证，单点登录异构系统
- 按职能维度定制门户（每个部室专属门户）
- 按信息维度定制门户（文本、图文、视频等多种内容形态）

**价值话术**：
- 从"分散找"到"聚合推"，实现业务找人
- 汇集信息、洞察数据、统一流程、融合业务、协同工作、千人千面

### 3.2 统一流程管理

**解决问题**：企业内部流程管理不规范，业务断点多，流程无闭环，运营效率低

**核心能力**：
- 强大的流程引擎（可视化表单引擎+流程引擎）
- 全面覆盖：人力资源、财务管理、行政管理、信息管理、销售和市场、供应链管理、质量管理、客户服务
- 统一流程中心（All IN ONE）：统一发起、待办/已办、待阅/已阅、监控、搜索
- 流程仿真审批、个性化开发平台
- 流程效率分析：全方位诊断审批效率、优化耗时瓶颈节点、辅助快速决策
- 流程融合一体化：协同流程与异构系统的对接管理平台

**价值话术**：
- 实现人事、行政、财务、业务流程标准化、数字化、规范化
- 突破传统职能流程视角，构建端到端流程闭环

**实施特点**：
- 流程实施0门槛
- 互联网级用户体验
- 开通融合整合能力强

### 3.3 统一知识管理

**解决问题**：重要文件保密性要求强，文件散落在个人电脑或云盘中，缺乏统一安全管理

**核心能力**：
- 知识创造、知识共享、知识学习、文控体系管理一站式服务平台
- 体系管理制度（文档新建、评审、归档、发布、更新全生命周期管理）
- 知识中心数据可视化
- AI智能知识赋能（智能学习、智能问答）
- 知识门户、知识检索、权限管理

**价值话术**：
- 管理更规范、协作更高效、记录可追溯、文件更安全
- 贯彻"ISO文控管理体系"
- 打造学习型组织

### 3.4 业务协同一体化（融合中心）

**解决问题**：传统OA与ERP集成需大量API开发，费用高、周期长、体验差

**核心能力**：
- **金蝶云深度融合**：
  - 单点登录、单据导航、快速查询
  - 流程单据融合（流程+ERP组装能力）
  - 业务插件引擎（无代码融合）
  - 群组业务协作（沟通中处理业务）
  - 协作组件（IM&单据融合）
- **融合中心架构**：数据源管理、基础资料融合、单据融合、实例数据融合
- **轻量业务场景**：轻采购、轻销售、轻资产、轻项目、轻车辆、轻差旅、轻费控、轻合同
- **全员应用场景**：出差申请、借款申请、费用报销、资产申请、采购申请、销售订单

**价值话术**：
- ERP一体化双向可组装式融合
- 让ERP业务具备沟通、会议、日程等协同能力
- 让ERP专业应用变成全员应用
- 0开发、配置级融合方案，又快、又好、又省钱
- 无需开发、实时数据、双向融合、全员应用

### 3.5 统一数据分析（BI报表）

**解决问题**：决策依赖经验判断而非数据分析，数据割裂，精细化管理难

**核心能力**：
- 数据源业务连接、快速实施能力、监控中心
- 业务数据分析、流程效率分析、运营分析、组织行为分析
- 管理驾驶舱（财务、人事、销售、采购、生产、仓储、能源、项目、质量等专题）

**价值话术**：
- 拉通ERP、CRM、OA等系统数据，精细化分析
- 为管理层经营决策提供数据支撑

### 3.6 信息安全管控

**核心能力**：
- 操作安全、登录安全、文件安全、群组安全
- 信息系统安全等保三级

### 3.7 平台引擎能力

8大引擎：
1. 轻云0代码应用搭建引擎
2. 融合&集成引擎
3. 组织引擎
4. BI引擎
5. 门户引擎
6. 表单&流程引擎
7. 内容引擎
8. 融合通讯引擎

开放能力：500+开放接口，支持异构系统快速集成

## 四、制造业场景痛点匹配

### 离散制造企业（机械加工、电子元器件、汽车、服装、家具、五金、医疗设备）

| 痛点 | 对应方案 |
|------|---------|
| 产品多样性、小批次、非标准化 | 流程管理+轻云应用快速搭建 |
| 库存压力大，资金周转压力大 | 融合中心+数据分析 |
| 委外加工管理力度弱 | 供应链协同流程 |
| 研发项目管理难度高 | 知识管理+项目协同 |
| 费用管控严格 | 费控一体化（OA+ERP融合） |
| ISO文控管理体系 | 统一知识管理 |

### 流程制造企业（化工、食品、饮料、制药、烟草）

| 痛点 | 对应方案 |
|------|---------|
| 大批量采购，信息协同要求高 | 门户+流程融合+数据分析 |
| 设备维保、备件管理标准高 | 轻云应用+流程管理 |
| 配方配料知识管控 | 知识管理（权限+体系化） |
| 缺乏标准化体系 | 流程引擎+文控体系 |
| 人员流失经营风险 | 知识管理+流程固化 |

## 五、客户案例库

### 华海通信（全球通信设备制造）
- **场景**：面向全球工程交付，研发制造多地协同
- **方案**：星空、s-HR、苍穹三应用集成在云之家；考勤、审批、知识中心、会议、信息中心统一管理
- **价值**：所有信息安全快速在一个门户里聚焦，提升业务和决策效率

### 舍得集团（流程制造-白酒）
- **场景**：多品牌矩阵，敏捷流程管理
- **方案**：75个分类、1228个模板、28个中心的流程管理数字中心
- **特色**：1个IT、无限落地的负责人制度

### 中电电气（离散制造-电气设备）
- **场景**：从SAP迁移到星空&云之家
- **价值**：销售审批效率提升210%、报价出错率降低98%、销售运营人工成本减少50%
- **方案**：一部手机实时准确开展销售业务（商机报备、报价、合同评审、数据分析）

### 引航生物（流程制造-生物医药）
- **方案**：金蝶云一体化业务决策分析平台
- **价值**：全面应用在财务、人事、销售、采购、生产、仓储、项目、质量、能耗等领域

### 石家庄众帮（离散制造）
- **场景**：PDA报工成本高、车间电脑少、质检协作效率低
- **方案**：掌上工厂（手机报工、生产领料、入库）
- **价值**：小投入大产出，数据实时推送，计件工资无差错

### 辽宁中蓝（离散制造-手机摄像头部件）
- **定位**：国家专精特新"小巨人"
- **方案**：研发一体化协同、费用报销一体化、非生产领料一体化
- **价值**：业务全员化、业务场景一体化

### 北京中科盛康（医疗器械-专精特新）
- **方案**：替换传统CRM，实现销售协同
- **价值**：敏捷一体化、星空深度融合、移动化办公

## 六、竞品对比要点

### vs 泛微/致远（传统OA）

| 维度 | 传统OA | 云之家 |
|------|--------|--------|
| ERP集成方式 | API/DB开发，费用高、周期长 | 配置级融合，0开发 |
| 架构 | 本地部署为主 | SaaS云原生，百万级并发 |
| 移动体验 | 移动端为辅 | 移动优先，互联网级体验 |
| 业务协同 | 流程审批为主 | 沟通+业务+流程深度融合 |
| ERP能力 | 无 | 与金蝶云天然一体 |

### vs 钉钉/企业微信/飞书

| 维度 | 互联网协同 | 云之家 |
|------|-----------|--------|
| ERP融合 | 无深度融合能力 | 金蝶云配置级双向融合 |
| 制造业适配 | 通用型，缺乏行业理解 | 深耕制造业场景方案 |
| 数据安全 | 数据在第三方平台 | 等保三级，私有化可选 |
| 流程引擎 | 轻量级 | 企业级流程引擎 |

### 关键差异化话术

1. **IDC市场第一**："IDC连续六年SaaS协同市场占有率第一，市场份额62.5%"
2. **ERP一体化**："0开发配置级融合，不是传统OA的API对接，是真正的业协一体"
3. **制造业专家**："13年服务10000+企业，大量制造业头部客户验证"
4. **成本优势**："传统OA集成ERP，光开发费可能就几十万，云之家配置即用"

## 七、"四化"价值总结

| 价值维度 | 内容 |
|---------|------|
| 一体化 | 一个平台、一个门户整合异构系统，应用一站式、数据分析一体化 |
| 协同化 | 敏捷搭建轻量级业务应用，OA+ERP业务协同化，群组高效业务协作 |
| 数治化 | 拉通ERP、CRM、OA数据，精细化分析，科学决策 |
| 智能化 | 数据+业务+智能技术结合，提升效率、降低成本 |
