# 0.2.14 版本说明

## 版本目标

- 优化用户 AI 端侧栏品牌区的图标与标题比例
- 将会话创建入口文案从默认英文切换为中文
- 继续收口用户侧工作台的正式产品观感

## 范围

- `apps/assistant-web` 侧栏品牌头部
- `apps/assistant-web` 会话创建按钮文案

## 关键页面 / 能力

- 用户 AI 端左侧品牌区
- 用户 AI 端会话列表创建入口

## 验收结果

- 用户侧品牌图标放大并增强视觉权重，标题字号与字重同步调整
- 侧栏头部图标与文字比例已向管理侧品牌效果靠拢
- `+ New chat` 已替换为 `新对话`

## 未完成项

- 暂未对用户侧其他导航字级和列表密度做统一收口

## 下一步计划

- 如需进一步统一双端品牌语言，可继续收口用户侧侧栏间距、会话项字级与顶部工具栏文案
