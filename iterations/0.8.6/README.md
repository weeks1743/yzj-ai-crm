# 0.8.6 公司研究图片生成注入个人 SOUL

## 版本目标

- 公司研究资料卡点击“生成图片”时，将当前用户个人 SOUL 注入图片生成 prompt。
- 用户保存或调整 SOUL 后，重新生成图片即可得到更贴近金蝶云之家制造业协同办公方案销售立场的视觉结果。

## 范围

- 仅修改 `assistant-web` 的图片生成 prompt 组装。
- 个人 SOUL 从现有个人设置接口读取后传入会话运行组件。
- 不改变 `/公司研究` 执行链路，不重新研究公司，不修改已有公司研究资料。

## 关键页面 / 能力

- AI 销售工作台：
  - 公司研究资料卡的“生成图片 / 重新生成图片”会带入 SOUL。
  - prompt 中明确要求把 SOUL 转化为销售视角、行业重点和价值主张，但不把 SOUL 原文排版进画面。
