# 0.8.7 公司研究图片生成取消 SOUL 注入

## 版本目标

- 将公司研究资料卡的“生成图片 / 重新生成图片”恢复为不关联个人 SOUL。
- 保留个人 SOUL 设置页、持久化能力和左下角用户身份展示。

## 范围

- 仅修改 `assistant-web` 图片生成 prompt 的组装方式。
- 不改变 `/公司研究` 执行链路。
- 不调整图片生成超时、存储、重试或 prompt 截断策略。

## 关键页面 / 能力

- AI 销售工作台：
  - 公司研究资料卡生成图片时，只使用公司名、资料标题、研究摘要和固定画面要求。
  - 请求 prompt 中不再包含 `销售 SOUL` 或 SOUL 原文。
  - 个人设置页仍可查看和保存 SOUL。

## 验收结果

- 已通过：`pnpm --filter @yzj-ai-crm/assistant-web build`

## 未完成项

- 本轮不新增图片持久化。
- 本轮不新增图片失败重试策略。
