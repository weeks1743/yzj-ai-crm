# 0.8.16 外部信息证据卡按资料聚合

## 目标

- 修复外部信息检索命中同一份公司研究多个片段时，用户端重复展示多张相同资料卡的问题。
- 保留“命中片段数量”语义，不把一个 Artifact 的多个 chunk 误当成多份资料。

## 范围

- 用户 AI 端证据展示区按 `artifactId + versionId` 聚合资料卡。
- 标题区显示“资料份数”和“引用内容片段数”。
- 每张资料卡保留最高相关度分数和代表性片段，继续支持查看完整研究、生成 PPT、生成图片。

## 验收

- 同一份公司研究命中 5 个片段时，只展示 1 张资料卡。
- 顶部仍提示 5 条引用内容，避免丢失检索命中信息。
- 不影响后端外部信息检索、回答内容和公司研究资料存储。
