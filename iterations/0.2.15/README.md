# 0.2.15 版本说明

## 版本目标

- 统一管理侧与用户侧的品牌标题文案
- 统一浏览器页签标题为 `AI销售助手`
- 使用现有共享 `logo.png` 作为双端网站图标

## 范围

- `apps/admin-pro` 运行时标题与页签图标
- `apps/assistant-web` 页面标题与页签图标
- `packages/shared` 品牌常量与文档 branding 工具

## 关键页面 / 能力

- 管理侧顶部品牌区与浏览器页签
- 用户侧侧栏品牌区与浏览器页签
- 双端共享品牌常量与 favicon 注入逻辑

## 验收结果

- 管理侧品牌标题已从 `AI销售助手管理侧` 统一为 `AI销售助手`
- 用户侧浏览器 title 已从英文标题统一为 `AI销售助手`
- 双端运行时都会把共享 `logo.png` 写入 favicon

## 未完成项

- 暂未扩展多尺寸网站图标导出，如 `favicon.ico`、`apple-touch-icon` 专用裁切版本

## 下一步计划

- 如需进一步完善品牌资产，可补充多尺寸 icon 产物并统一浏览器兼容策略
